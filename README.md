# 🌱 AgroEquilíbrio - Agrinho 2026

Este projeto digital foi desenvolvido por **Sofia Figueiredo Silva** para o concurso **Agrinho 2026**. O objetivo do site é conscientizar sobre a importância da união entre a produtividade agrícola e a conservação ambiental.

---

## 🧮 Calculadora de Diagnóstico Rural

O ponto central do site é uma ferramenta interativa que permite ao usuário simular o nível de sustentabilidade de uma propriedade rural com base na relação entre área de produção e preservação nativa.

A calculadora funciona de forma simples:

1.  **Entrada de Dados:** O usuário define dois parâmetros básicos da fazenda:
    * **Área Total (ha):** O tamanho da propriedade em hectares.
    * **Árvores Preservadas:** A quantidade de árvores nativas mantidas na propriedade.
2.  **Processamento:** Ao clicar no botão de análise, o site processa esses dados instantaneamente.
3.  **Resultado:** A ferramenta gera um **Índice de Sustentabilidade** em porcentagem (%), que "corre" de 0 até o valor final. O resultado é acompanhado de um status (como "Excelente", "Bom" ou "⚠️ Atenção") e uma breve recomendação personalizada sobre o equilíbrio ambiental da propriedade.
